# /cabelma-level-up

Encontrá y arrancá una nueva automatización. Una por semana.

**Cuándo correr:** los viernes, o cuando querés encontrar la próxima mejora.

Usa el Marco CAI (`references/marco-cai.md`): Cabeza → Acción → Infraestructura.

---

## Paso 1 — Cabeza: encontrar

```
1. ¿Qué tarea manual repetiste más veces esta semana?
2. ¿Hubo algo que te hizo pensar "esto debería ser más fácil"?
3. ¿Hubo algo que llegó tarde o te olvidaste por ser manual?
```

---

## Paso 2 — Acción: elegir

Si hay más de una candidata:
```
A) Si esto funcionara mañana, ¿cuánto tiempo te ahorraría por semana?
B) ¿Qué tan difícil parece? (fácil / medio / complejo)
```
Elegí la de mejor ratio impacto/dificultad.

---

## Paso 3 — Infraestructura: arrancar

```
1. ¿Cómo es el proceso hoy, paso a paso?
2. ¿Qué entra? (datos, archivos)
3. ¿Qué tiene que salir?
4. ¿Dónde querés que viva? (Excel, script, AI, etc.)
```

Generá una propuesta concreta siguiendo el Principio Lego: una función por paso.

---

## Entregable

Producí exactamente uno:
- **Prompt reutilizable** — para pegar en el AI cada vez
- **Estructura de script** — pseudocódigo o estructura básica
- **Checklist/SOP** — proceso estandarizado si no es automatable aún

---

## Cierre

```
Candidata: {nombre}
Tipo: {Prompt / Script / SOP}
Ahorro estimado: {tiempo/semana}
Estado: {listo / en progreso / próxima semana}
---
{el entregable}
---
```

Guardalo en `references/sops/`.

**Notas:**
- Una automatización por sesión.
- Método Bici: al 80% y funcionando > al 100% en desarrollo.
- Si el usuario dice "no hay nada", revisá `context/schedule.md` — siempre hay algo.
